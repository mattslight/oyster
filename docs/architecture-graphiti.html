How data flows
1
You chat in the Tokinvest space → conversation saved as episode (group_id=tokinvest) → entities extracted → "Bharat → works_on → token design" added to graph
2
Next session in Tokinvest → OpenCode calls search_facts with group_id → relevant entities and relationships injected into prompt → AI has full context
3
You say "budget review moved to April 3rd" → Graphiti invalidates old date with timestamp → new fact has validity window → historical queries still work
4
You ask "who's involved in Tokinvest and what's blocked?" → graph traversal: Bharat → works_on → token design → blocked_by → legal review → structured answer
5
Gmail connector (future) syncs emails → each email becomes an episode → entities extracted → deadlines, contacts, decisions flow into the graph automatically
6
Cross-space query → "what needs my attention?" → searches across all group_ids → Oyster synthesises priorities from KPS + Tokinvest + Personal

Sessions filter — UX options

Five interactive variants. Same sample data in each (4 active · 1 waiting · 2 disconnected · 16 done — 23 total). Click around. The current production behaviour is option 0: single-select, "all" includes everything.

filter-options.html

CURRENT Single-select including done

What ships today. Click any chip to filter to that one state. "All" includes everything (done is bundled in the count). Done sessions are mixed into the inventory by default.

OPTION 1 Hide done by default · toggle to includemy pick

Done is treated as archive. Coloured chips behave exactly like today (single-select). "All" means "all-not-done". A separate + N done toggle on the right adds done back in. Most people never click it.

Try: click "all" — see done are hidden. Toggle "+ 16 done" — they appear. Click red chip — only disconnected.

OPTION 2 Multi-select toggle chips (Gmail/GitHub model)

Each chip toggles independently. None selected = show all. "All" toggles all on/off. To get "active+waiting+disconnected" — click those 3, or click "all" then click "done" to remove it.

Try: click red, amber, green individually. Click "all" — all light up. Click "done" while all are on — done excluded.

OPTION 3 Modifier-key multi-select

Single click = single-select (current). Cmd+click (or Ctrl on Windows) = add/remove from selection. "All" clears the filter. Power-user move; not discoverable on its own.

Try: click red. Cmd+click amber and green to add them. Cmd+click red again to remove it.

OPTION 4 "Live" preset chip

An extra chip "Live" pre-bundles active+waiting+disconnected. Single-select on everything (you'd switch between "Live", "Active", "Done" etc). One-click for the common case but adds a pseudo-state.

Try: click "Live" — green/amber/red rows shown, done hidden. Click red individually — only disconnected.

OPTION 5 "≥ this liveness" range select

Chips are ordered active → waiting → disconnected → done. Click "disconnected" → shows everything at least this live (active + waiting + disconnected). Click "active" → only active. The natural ordering does the work.

Try: click red — green/amber/red rows shown. Click amber — active+waiting only. Click "all" — full list with done.

What changed vs. the previous mock:
  • No "Spaces" content section on Home. Pills enumerate the spaces; doing it twice was redundant.
  • Pills carry multiple pips — one per non-zero active / waiting / disconnected count. Done is omitted (resting state, would always show). Hover any pill for the full count breakdown.
  • + create space moves to the end of the top nav. Mirrors how the chat-bar pills work.
  • On a real-space view, Projects is the first content section — children of the current scope.

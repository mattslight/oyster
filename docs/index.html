Built on MCP. Bring Your Own Agents.

Oyster is an MCP server. That means any agent that speaks the protocol — Claude Code, Cursor, VS Code, Windsurf, or your own — can control your workspace. Create documents, organise spaces, open artefacts. One standard, every agent.

Detailed setup →
terminal
claude mcp add --scope user --transport http oyster http://localhost:4444/mcp/
.cursor/mcp.json
{
  "mcpServers": {
    "oyster": {
      "url": "http://localhost:4444/mcp/"
    }
  }
}
.vscode/mcp.json
{
  "servers": {
    "oyster": {
      "type": "http",
      "url": "http://localhost:4444/mcp/"
    }
  }
}
~/.codeium/windsurf/mcp_config.json
{
  "mcpServers": {
    "oyster": {
      "serverUrl": "http://localhost:4444/mcp/"
    }
  }
}

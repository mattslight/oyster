Capabilities

Everything in one surface

No more tab-switching between terminals, browsers, docs, and dashboards. Oyster OS brings them all into a unified desktop metaphor.

Live Terminal

A real PTY process on the server, streamed over WebSocket to xterm.js. Full color support, scrollback replay on reconnect, live resize.

AI Chat

Integrated assistant with SSE streaming. Create sessions, send messages, watch responses arrive in real-time through the embedded chat bar.

App Management

Start, stop, and monitor dev servers from the desktop. Live status indicators with automatic port detection and health checks.

Doc Viewer

Render Markdown and HTML inline with glassmorphic windows. Navigate between docs, go fullscreen, and browse with prev/next controls.

Workspaces

Organize projects into named spaces. Each space groups its apps and docs, with instant switching via the persistent pill navigation.

Glassmorphic UI

Draggable windows with backdrop-blur and glow effects. Aurora background shader and smooth animations throughout.

Info Architecture
Under the hood

Built to be fast and simple

No bloated frameworks on the server. A single Node.js process handles HTTP, WebSocket, and process management. The client is React + Vite with zero routing overhead.

Server
Node.js
Frontend

WS + SSE

Zero-framework server

Raw node:http + node:ws. No Express, no Fastify. Just a request handler and a WebSocket server. Under 400 lines total.

Window manager

Draggable, stackable windows with z-index management. Viewer windows support fullscreen with floating toolbar navigation.

Process lifecycle

Start and stop dev servers by name. Port detection via TCP connect, HTTP readiness checks, and automatic cleanup on exit.

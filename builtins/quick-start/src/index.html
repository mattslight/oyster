Quick Start.

Everything you need to know in 60 seconds.

The prompt bar

Type anything into the prompt bar to talk to Oyster. Ask it to create documents, organise your work, or answer questions about your projects. Any printable key focuses the prompt bar automatically.

Slash commands
/s <name> Switch space. Fuzzy matches. /s bf finds "blunderfixer".
/o <search> Open an artifact by name. /o pitch opens your pitch deck.
#1 #2 #3 Jump to a space by position. #. for home, #0 for all.
Cmd+K Spotlight search across all artifacts.
Spaces

Spaces are workspaces — one per project, client, or topic. Switch between them using the pills below the prompt bar, or with #name and /s name.

Click + to add a new space, or drop a folder anywhere on the surface to import a project automatically.

Artifacts

Everything on the surface is an artifact — notes, decks, diagrams, apps, tables, wireframes. Click an icon to open it. The topbar (hover near the top) has sort, filter, and view controls.

Drop to import

Drag a folder like ~/Dev onto the surface. Oyster scans it, finds projects, and creates spaces for each one.

The lightning bolt

Click the bolt icon in the prompt bar to open the shell. Full terminal access — talk directly to the AI engine with no guardrails.
